Tarea de Algoritmos computacionales II

--Ojetivos de los programas--

Definir las siguientes funciones que devuelvan los siguientes resultados:

• Función que reciba por parámetro 2 números enteros devuelva la suma de dos
números.

• Función que reciba por parámetro la edad y devuelva un mensaje indicando si
es mayor o menor de edad.

• Función que reciba por parámetro el radio y devuelva el área de un círculo.

• Función que reciba por parámetro la base y la altura y devuelva como resultado,
el área de un triángulo.

• Cuantos son pares.
• Cuantos son impares.
• Cuantos son mayores a 10.
• Cuantos son positivos.
• Cuantos son negativos.
